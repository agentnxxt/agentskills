How to Install & Use
Autonomyx Skills

From download to your first AI-powered output — a complete guide for getting started with AgentNXXT skills in Claude.

Part 1 — Installation
1
Choose a skill from the marketplace

Visit the Skills Marketplace and browse the available skills. Each card shows what the skill does, which category it belongs to, and what topics it covers.

2
Click ⬇ Download on the skill card

Each skill card has a purple ⬇ Download button. Clicking it directly downloads a .skill file to your computer — no GitHub account needed.

💡
What is a .skill file? It's a small package containing all the instructions Claude needs to become an expert in that domain. Think of it like an app you install — except for Claude's brain.
3
Open Claude.ai and go to Settings

Go to claude.ai and click your profile icon in the top-right corner. Select Settings from the dropdown menu.

claude.ai → Profile icon → Settings
4
Navigate to the Skills tab

In Settings, find the Skills section in the left sidebar. This is where all your installed skills live.

Settings → Skills
⚠️
Don't see Skills? The Skills feature may be rolling out gradually. If you don't see it yet, check back in a few days or make sure you're on a Claude Pro or Team plan.
5
Upload the .skill file

Click Upload skill (or the + button) and select the .skill file you downloaded in Step 2. Claude will install it instantly.

✅
You can install multiple skills. Each skill is independent — install as many as you need. They won't conflict with each other.
Part 2 — Using a Skill
6
Start a new conversation — no commands needed

Skills activate automatically. You don't need to type a special command or mention the skill name. Just describe what you need naturally and Claude will detect when a skill applies.

"Repurpose this blog post for LinkedIn for a C-suite fintech audience"
"Compare the TCO of Notion vs Confluence for a 50-person team"
"Draft a privacy policy for my healthcare SaaS"
7
Review the output and give feedback

After every response, Claude will ask if you're satisfied with the output. You can:

👍 Approve ✏️ Request changes 🔄 Ask for a variation

Claude will iterate up to 3 times based on your feedback. If you're still not happy after 3 rounds, it will offer to connect you with a human expert via a booking link.

8
Check for updates periodically

Skills are actively maintained and improved. Claude will automatically notify you at the start of a session if a newer version of your skill is available:

⚠️ Update Available — You're on v1.0.0, latest is v1.0.1. 👉 Get the latest: github.com/agentnxxt/agentskills

To update, simply download the latest .skill file and re-upload it in Settings → Skills. Takes under a minute.

FAQs
Do I need a paid Claude plan?

The Skills feature is available on Claude Pro and Team plans. It may not be available on the free tier.

Why can't skills auto-install like official Claude skills?

Official skills (like docx, pdf) are reviewed and listed by Anthropic — it's a closed process. Autonomyx skills are community-built and user-installed for now. We're working toward getting listed in Anthropic's official directory.

Can I use multiple skills at once?

Yes — install as many as you need. Claude manages them independently and activates the right one based on your request.

Are the skills free?

100% free and open source. You can also fork, modify, and contribute back on GitHub.

Something not working?

Open a GitHub issue or share feedback via the form. For complex help, book a 15-min call with Chinmay.
